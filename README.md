# Aplikasi Cuti Babinsa

Login dengan NRP, data nama & pangkat otomatis dari database master.  
Fitur: Tambah/Edit/Hapus cuti, download PDF (semua/satu), TXT (JSON), CSV.

## Setup
1. Buat tabel di Supabase (`master_anggota` dan `permohonan_cuti`)
2. Ganti `SUPABASE_URL` dan `SUPABASE_ANON_KEY` di `script.js`
3. Buka `index.html` di browser
